Manual Test: Visit http://localhost:3000/login

  1. Go to http://localhost:3000/login
  2. Click "Log In" button without filling anything
  3. Check if "Email is required" error appears
  4. If not, open DevTools and check console for errors

Expected Behavior:

Validation error "Email is required" should appear below the email field when clicking submit with an empty form.

Actual Behavior:

Based on Playwright test results, no validation error appears.

Possible Issues:

  • UForm doesn't validate on submit for untouched fields (added validateOn="['input', 'blur', 'submit']")
  • Validation errors not rendering in DOM
  • UFormField not displaying error property
